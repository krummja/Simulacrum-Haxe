# Simulacrum Dev Notes

I had the idea to use class Pokemon-style world tiles, but in an open-world RPG with editable environments. I loved the Pokemon Ruby/Sapphire secret base concept, but I want to expand it into an almost Dwarf Fortress type of system where the player can modify the terrain arbitrarily.

I want magic and other effects to be able to make massive changes to the terrain, set fires, dig holes, carve mountains, etc.
